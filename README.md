# Smart FAQ Assistant 🤖

A chatbot built using Python, Streamlit, and NLP techniques.

## Features
- FAQ-based chatbot
- TF-IDF text vectorization
- Cosine similarity matching
- Chat history
- Clear chat button
- Unknown question detection
- Streamlit web interface

## Technologies Used
- Python
- Streamlit
- Scikit-learn
- JSON

## NLP Techniques
- TF-IDF Vectorization
- Cosine Similarity
